Use case: OWL profiles

Description
Angela is an ontology designer, after decades of work she has accumulated a long list of ontologies on different domains using a variety of OWL profiles. She had the idea to give back to the community by creating a resource where to find these ontologies and request them with a specific OWL profile, for example EL, QL.
Requirements
  • The client needs to convey the needed OWL profile.
  • The server needs to adapt the ontology if possible.
Solutions
  • Profile negotiation to convey OWL profile.
  • Use the URIs provided in [L1] for OWL profile identification.
Content negotiation dimensions
  • Profile.
  • OWL Profile.

Use case: Capability matching

Description
John have multiple mobile devices each with its own capabilities, one device can render color content while others can not, also each device has a different size, and is able to consume some formats.
Requirements
  • Serve different representations according to the device capabilities.
Solutions
  • Use CC/PP as a mean to request capability constraints.
Content negotiation dimensions
  • Capability.

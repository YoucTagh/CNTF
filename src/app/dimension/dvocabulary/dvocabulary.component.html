Dimension: Vocabulary

Description
The Resource Description Framework (RDF) is a general language for representing information on the Web. This information on the Web can be described using different vocabularies. A vocabulary is a data model with classes, properties and relationships that can be used to describe data and metadata. Users may want to request representations that are described with a certain vocabulary, e.g. data about person described by FOAF or vCard.

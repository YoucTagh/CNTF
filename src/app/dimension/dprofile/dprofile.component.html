Dimension: Profile

Description
One definition of a profile is "a description of structural and/or semantic constraints on resource representations that apply in addition to the constraints inherently indicated by their MIME type". A user may wish to receive a representation only if it conforms to a particular profile, such as the DCAT application profile.

Dimension: Entailment Regime

Description
Various W3C standards, including RDF and OWL, provide semantic interpretations for RDF graphs that allow additional RDF statements to be inferred from explicitly given assertions. An entailment regime specifies how an entailment relation can be used to redefine the evaluation of basic graph patterns from a SPARQL query using the SPARQL extension point for basic graph patterns matching. A user may want to request data that conforms to a shape but after using a specific entailment relationship, such as an RDF Schema or RIF entailment.

Action Plan

To create this resource we followed this action plan:

  • Define a content negotiation use case: title, description and challenges.
  • Extract from the use case content negotiation dimensions, style
  • Gather references of materials: articles, technical reports, documentations of implementation, empirical study. That tried to tackle some aspect of the use case, the search is done using [1], [2], [3].
  • Extract the solutions if any that are proposed in the references
  • To confirm validity and reliability:
    • As selection criteria, privilege references with higher citation number because it indicates relevance .
    • Discuss with supervisors and colleagues.
    • Collect and analyze peer feedback sent through the feedback form.

[1]: Google Scholar.

[2]: The DBLP Computer Science Bibliography.

[3]: Semantic Scholar.
